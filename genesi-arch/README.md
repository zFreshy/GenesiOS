# Genesi Arch - Build Files

Este diretório contém os arquivos necessários para construir a ISO do Genesi OS Arch Edition usando archiso.

## Build Rápido

```bash
sudo ./build-genesi-arch.sh
```

A ISO será gerada em `out/`.
